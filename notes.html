Qu'est ce que JQuery C'est une bibliothèque rapide et légère qui simplifie l'interaction avec le html, le css et le javascript sur une page web. Avantages : Simplifie la manipulation du DOM Génère les évènements de manière élégante Failite les animations et les effets Assure un dynamisme au site Inclusion ou utlisation Pour utiliser jQuery, on télécharge la bibliothèque depuis jquery.com ou on utilise une version hébergée puis on l'insère dans le HTML.
